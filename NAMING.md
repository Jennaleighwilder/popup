# Naming ideas (Popup may be taken)

**Popup** is a common word — event pop-ups, browser pop-ups, retail pop-ups. Consider these alternatives:

## Strong options
- **Popupify** — distinct, brandable, .com likely available
- **Pop Event** / **PopEvent** — clear, memorable
- **Fête** — French for "party," elegant, event-focused
- **Soirée** — already a theme name; could work as product name
- **Gather** — warm, community feel (check availability)
- **Lively** — energetic, event-appropriate

## Event-specific
- **Event Pop** — descriptive, SEO-friendly
- **Pop Page** — short, punchy
- **Page Pop** — same idea, different order

## Check availability
Before committing: domain (.com, .so, .io), Twitter/X, Instagram, App Store, trademark search.
